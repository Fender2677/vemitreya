# Changelog

Все значимые изменения в проекте. Формат основан на

## [2.206] — 2026-05-28

Первый публичный релиз.

### Возможности

- **Дашборд**: метрики системы (CPU/RAM/диск), живой трафик, активные каналы и
  маршрутизация трафика.
- **Переключение каналов**: ручной выбор сервера и умный автовыбор быстрейшего
  с настройками на каждую группу — интервал проверки, порог переключения,
  исключения серверов по ключевым словам. Списки серверов в каналах сворачиваются.
- **Группы прокси**: визуальный редактор с разделением на маршрутизацию и каналы.
  Выбор отдельных серверов из подписок через `use` + `filter`. Удаление канала
  с автоочисткой ссылок в группах и правилах.
- **Подписки** (proxy-providers): добавление и обновление.
- **Правила** (rules): редактор с изменением порядка и фильтрацией соединений.
- **Speedtest**: матрица «прокси × сайты».
- **AmneziaWG**: создание туннелей и интеграция в Mihomo, нормализация конфигов.
- **TrustTunnel**: обнаружение и регистрация SOCKS5-туннелей.
- **Роутеры**: генерация скриптов автонастройки для MikroTik (RouterOS 6/7) и
  Keenetic с экспортом списков доменов/IP.
- **Mihomo YAML**: встроенный редактор конфига с сохранением комментариев.
- **Обновления**: Mihomo и панель через архив или GitHub-релизы.
- **Telegram**: уведомления о падении сервисов.

### Технические детали

- Backend на FastAPI, frontend на React без сборки.
- Умный автовыбор обходит ограничение нативного Mihomo url-test (не работает
  поверх вложенных групп): панель пингует серверы напрямую и переключает сама.
- Редактирование YAML сохраняет комментарии (ruamel.yaml).
